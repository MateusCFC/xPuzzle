# xPuzzle

Implementação do jogo [15-puzzle](https://en.wikipedia.org/wiki/15_puzzle), em Flutter.

Possui quatro níveis de dificuldade:
 - Fácil (3x3)
 - Normal (4x4)
 - Difícil (5x5)
 - Hardcore (6x6)
